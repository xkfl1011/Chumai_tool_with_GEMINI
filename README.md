## CHUNITHM

A simple tool to extract your CHUNITHM play counts from the official CHUNITHM-NET and visualize them on a local web dashboard.

Features
* **Data Extraction:** Extracts total and difficulty-specific (BASIC to ULTIMA) play counts from your CHUNITHM-NET music records.
* **JSON Export:** Automatically saves the scraped data as a structured JSON file.
* **Web Dashboard:** Provides a local HTML dashboard featuring an interactive, sortable ranking table with expandable difficulty details.

Usage
1. Log in to [CHUNITHM-NET](https://chunithm-net-eng.com/mobile/home/) and navigate to the **Music Record (By Music)** page.
2. Open your browser's Developer Tools (F12) -> **Console**, paste the extraction script, and press Enter to download your JSON data.
3. Open `chunithm_songs~~~.html` in your web browser and upload the JSON file to view your personalized ranking.

Tech Stack
* Vanilla JavaScript (ES6+)
* HTML5 / CSS3
* **GEMINI**

Notices
* This is a fan-made project and unofficial tool and is not affiliated with SEGA.
* Trying to extract all songs (1,400+) at once may cause a session timeout. It is highly recommended to filter the song list by **Genre** or **Level** before running the script.
